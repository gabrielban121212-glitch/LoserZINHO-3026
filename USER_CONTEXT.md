# Braains.io Website - User Context

## About Braains.io
Braains.io is a multiplayer zombie survival .io game where players battle in a zombie apocalypse. Players can be humans trying to survive or zombies trying to infect others. It's a team-based tag game with up to 52 players per server.

## Key Information:
- **Game Type**: Multiplayer zombie survival game
- **Gameplay**: Team-based (Humans vs Zombies)
- **Created by**: m0dE on modd.io platform
- **Community**: 30,000+ Discord members
- **Versions**: Classic 2D and new 3D version available

## Features to Highlight:
1. **Multiplayer Action** - Up to 52 players per server
2. **Survival Mechanics** - Pick up items, buy equipment, strategic gameplay
3. **Community** - Active Discord with challenges and events
4. **Easy Controls** - WASD movement, simple item mechanics
5. **Cross-platform** - Playable in browser
